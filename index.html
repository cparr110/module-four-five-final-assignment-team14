COMP 617 final project

Can color bias how people read energy data?

This page combines the preliminary visualization and the survey comparison dashboard into one GitHub Pages entrypoint. The experiment tests whether semantically loaded colors shift viewers' sentiment, narrative framing, and policy attitudes when the data itself stays constant.

Preliminary design

Original preliminary chart

This chart is kept visually consistent with the original index page. The surrounding page is redesigned for GitHub Pages, but the graph itself uses the same D3 code, dimensions, labels, and color mapping as the original.

Experiment design

Same data, different color semantics

Participants saw one version of the visualization and answered factual, interpretive, policy, and debrief questions. The controlled variable was the color mapping: red and green were used to test whether culturally loaded colors changed interpretation without changing the underlying numbers.

Survey 1Alarm condition

High-connotation colors: red and green make the energy source contrast feel evaluative.

Survey 2Cool/neutral condition

Blue-based palettes reduce the obvious danger/safety signal while preserving readability.

Measured outcomesAccuracy + sentiment

Factual accuracy, perceived outlook, policy support, confidence, and open-ended reactions.
